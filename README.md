# Real-Time Trading App

A Flask-based real-time price prediction trading application that provides live market data, multiple technical indicators, and Buy/Hold/Sell signals.

## Features

- **Live Market Data:** Fetches up-to-date information for stocks and cryptocurrencies.
- **Technical Indicators:** Calculates Moving Averages, RSI, MACD, and Volume data.
- **Trading Signals:** Generates Buy/Hold/Sell recommendations alongside confidence scores and risk levels.
- **Dynamic Dashboard:** Displays information and visually rich charts that update seamlessly.
- **Advanced Search:** Includes autocomplete suggestions, asset type toggles (Stocks/Crypto), and country filtering.

## Technologies Used

- Python (Flask)
- HTML/CSS/JavaScript

## Getting Started

1. Clone the repository.
2. Install the required packages (e.g., using `pip install -r requirements.txt` if available).
3. Run `app.py` to start the Flask server.
4. Access the application in your local web browser.
